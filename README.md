# BCM02 JavaScript 自動化腳本

這是一個用於自動化執行 BCM02 報表填寫流程的 JavaScript 腳本。

## 功能簡介

本腳本主要用於自動化處理客戶互動紀錄的填寫與簽核流程，包含：
1.  自動搜尋「本月未檢視戶數」。
2.  循環處理客戶列表。
3.  自動點擊「更多」並新增互動紀錄。
4.  自動填寫表單內容（客戶來源、聯繫管道、聯繫結果等）。
5.  自動送出並關閉視窗，接續處理下一位客戶。

## 執行方式

1.  複製 `BCM02_automation.js` 的完整程式碼。
2.  在瀏覽器中打開要自動化執行的報表頁面。
3.  打開開發者主控台 (Console)：
    *   快速鍵：`F12` 或 `Ctrl` + `Shift` + `I`
    *   或點選瀏覽器選單中的「更多工具」->「開發人員工具」，然後切換到 "Console" 分頁。
4.  在 Console 中貼上程式碼 (`Ctrl` + `V`)，然後按 `Enter` 執行。

## 疑難排解

### 出現「已在偵錯工具中暫停」(Paused in debugger)

如果在執行過程中程式停住，並顯示「已在偵錯工具中暫停」，通常是因為 Chrome 開發者工具開啟了 **「發生例外狀況時暫停」** 的功能。這會導致網頁原本的程式碼（非本腳本）拋出任何錯誤時，瀏覽器強制中斷執行。

**解決方法：**

1.  在開發者工具中，找到「停用中斷點」按鈕（圖示通常是一個被劃掉的六角形或類似標誌），或按快速鍵 `Ctrl` + `F8`。
2.  接著按「繼續執行指令碼」按鈕（播放鍵圖示），或按快速鍵 `F8`。
3.  腳本應會繼續執行。

> 建議在執行腳本前，先確認開發者工具的「發生例外狀況時暫停」功能已關閉 (圖示應為灰色/未啟用狀態)。（乙仔：這句話我看不懂，都是發生中斷時才去按「Ctrl + F8」和「F8」）
